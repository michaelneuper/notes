#+title: Trigonometric Functions

* Definition
- A function \(f\) is said to be trigonometric if it is one of the following:
  - \(f(x) = \sin(x)\)
  - \(f(x) = \cos(x)\)
  - \(f(x) = \sec(x)\)
  - \(f(x) = \csc(x)\)
  - \(f(x) = \tan(x)\)
  - \(f(x) = \cot(x)\)
