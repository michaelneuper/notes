#+title: Integers

* Values
- Python3 only limited by memory
- Python2 has int (2^64-1) and long (arbitrary)

* Typical Literals
- 1234
- 0
- 99

* Operations
- Addition
- Subtraction
- Multiplication
- Quotient
- Remainder ([[id:a5a2ca3a-f512-41a4-a782-8ec27c0bfe44][Modulo]])
- Exponentiation

* Operators
- =+=
- =-=
- =*=
- =//= *NOTE*: Different from =/= with [[id:321d94dc-4d0a-479e-a0e8-8f5603f7915e][floats]]
- =%=
- =**=

* Examples
- =1 + 2= -> =3=
- =7 // 3= -> =2=
