#+title: Clairaut's Theorem
#+filetags: theorem

* Definition
Suppose \(f\) is defined on a disk \(D\) that contains the point \((a,b)\).
If the functions \(f_{xy}\) and \(f_{yx}\) are both continuous on \(D\), then
\begin{equation*}
f_{xy}(a,b) = f_{yx}(a,b)
\end{equation*}
